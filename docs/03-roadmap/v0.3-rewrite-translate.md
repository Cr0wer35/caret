# v0.3 — Rewrite + Translation (directional)

_Last updated: 2026-04-21_

**Status**: Backlog. Not started.

## Goal

Add two **selection-triggered** features that complement the streaming core of v0.1/v0.2:

1. **Rewrite** — user selects text, invokes Caret via shortcut, picks a tone (professional, concise, friendly, casual, simpler), replacement streams in.
2. **Translate** — user selects text, invokes Caret via shortcut, picks a target language, translation streams in.

Both features use a different UX pattern than v0.1/v0.2: the user **explicitly invokes them on selected text**, rather than having Caret watch while typing. This is the one place where Caret adopts a pattern similar to Apple Intelligence Writing Tools — because it fits the task better.

## Why selection + shortcut, not streaming

- Rewrite and translate are **intentional operations**, not corrections. The user must decide when to ask.
- Streaming them while the user types would produce constant distractions.
- Selection scope lets the user target a specific chunk, which is essential for both features.

## Interaction

- Shortcut (default `⇧⌥⌘C`) triggers a small popover near the selection.
- Popover shows a tone or language picker.
- Pick → request fires → replacement streams in.
- `Tab` or click "Accept" commits; `Escape` cancels.

## Likely milestones (to be refined)

- **R1** — Selection capture via AX (read `kAXSelectedTextAttribute` + `kAXSelectedTextRangeAttribute`).
- **R2** — Shortcut-invoked popover UI.
- **R3** — New prompt files: `RewritePrompt.v1.md` (with tone parameter) and `TranslatePrompt.v1.md` (with target language).
- **R4** — Accept / cancel flow that replaces selection via AX.
- **R5** — Tone and language defaults configurable in settings.
- **R6** — Dogfood + release v0.3.

## Open questions

- **Tone list**: what defaults? `professional`, `concise`, `friendly`, `simpler`. User-editable?
- **Language list**: start with FR, EN, ES, DE, IT, PT? Auto-detect source?
- **Undo**: `Cmd+Z` must cleanly restore the original selection. Critical for trust.
- **Should rewrite work on unselected text?** E.g. "rewrite this paragraph" where paragraph is inferred. Probably no — too much surprise.

## Non-goals

- No multi-step rewrite (rewrite then refine). Single shot.
- No custom tones user-defined via prompt. Could be v0.4.
- No document-level translation (only selection).
